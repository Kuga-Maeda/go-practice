# Go学習記録（初日）

## 学習日
2025/11/3

## 学習内容

### 1. Go言語基礎
- 変数、型、定数の理解
- 条件分岐（if文）、繰り返し（for文）
- 関数の定義と呼び出し
- 値渡しとポインタ渡しの理解

### 2. 構造体とメソッド
- `struct` の定義
- メソッドの追加（値レシーバ・ポインタレシーバ）
- メソッド呼び出しでフィールドを操作

### 3. インターフェース
- インターフェースの定義
- 異なる型（Person、Dog）を同じインターフェース型で扱う
- 共通の振る舞いを抽象化する方法を学習

### 4. スライス・マップと関数
- スライスを関数に渡して合計計算
- マップを関数に渡して全要素出力
- データ構造の操作と関数設計の練習

### 5. エラー処理
- 戻り値によるエラー処理 (`error`)
- `panic` と `recover` を使った安全な例外処理

## 学習環境
- MacBook (zsh)
- VSCode
- Go 1.21.x
- GitHub でコード管理

## 所感
- 基礎文法から関数・構造体・インターフェースまで一通り体験
- 関数でのエラー処理や panic/recover で Goの特徴を理解
- GitHubに記録することで振り返りやすく、今後の応用学習に備えられる
